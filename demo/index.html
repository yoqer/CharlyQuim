🤖 AI Feedback Widget Demo

Welcome!

This is a demo page showing the AI Feedback Widget in action.

Look for the chat button in the bottom-right corner! Click it to:

  • Ask questions about documentation
  • Report a bug
  • Give product feedback

How to embed on your site

Add this snippet before the closing </body> tag:

<script src="http://localhost:3000/widget.js"></script>
<script>
  AIFeedbackWidget.init({
    botId: 'default',
    apiUrl: 'http://localhost:3000'
  });
</script>

Configuration Options

Customize the widget appearance and behavior:

AIFeedbackWidget.init({
  botId: 'your-bot-id',          // Required
  apiUrl: 'https://api.yoursite.com', // Required
  position: 'bottom-right',       // 'bottom-right' or 'bottom-left'
  primaryColor: '#3b82f6',        // Any hex color
  icon: 'https://...'             // Custom icon URL
});
